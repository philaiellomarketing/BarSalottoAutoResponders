🔌 Connect Google Business Profile

This connects the dashboard to your Google reviews so it can pull new reviews and post your approved replies automatically. Step 1 is the slow part — start it today.

1 · Request API access (do this first — Google takes several days)

  • Go to console.cloud.google.com and create a project (e.g. "Bar Salotto Responders"). Use the Google account that manages the Bar Salotto Business Profile.
  • Enable these three APIs: Google My Business API, My Business Account Management API, My Business Business Information API.
  • Submit the access request form: support.google.com/business/contact/api_default — paste your Project ID, use case: "Reading and replying to our own restaurant's customer reviews."

2 · Create OAuth credentials

  • Cloud Console → OAuth consent screen: type External, app name "Bar Salotto", add yourself as a Test user, scope business.manage.
  • → Credentials → Create OAuth client ID → Web application. Authorized redirect URI: https://developers.google.com/oauthplayground. Save the Client ID and Client secret.

3 · Get a refresh token

  • Open OAuth Playground → gear icon → check "Use your own OAuth credentials" → paste Client ID + secret.
  • In "Input your own scopes" enter https://www.googleapis.com/auth/business.manage → Authorize → sign in with the business-owner account → Allow → "Exchange authorization code for tokens" → copy the Refresh token (starts with 1//).

4 · Send me the three values

Message me the Client ID, Client secret, and Refresh token — I'll walk you through saving them securely in Cloudflare (a few clicks in the dashboard, no coding) and flip the live connection on. Don't post them anywhere public.

Until Google approves Step 1, this dashboard shows manually-entered reviews — that's expected, not a bug.
